# TemplateString

A simple string templating library for Chapel. It allows you to define a
template string with placeholders for variables, and then replace those
placeholders with actual values from a map.

## Installation

Add TemplateString as a Mason dependency:

```bash
mason add TemplateString@0.2.0
```
